Raw data downloaded from CAMDA 2017 Neuroblastoma Data Integration Challenge. See http://www.camda.info.

Processed data provided in "processed" folder; these are obtained as outlined in the paper. See run_processing.R file in rna_seq_data_scripts folder for details. 

Clinical data stored in "patient_data" file. See patient_data.R file in patient_data_scripts folder for details.

"GeneCards.csv" obtained from GeneCards database on May 11, 2017.

"kegg_map_id_name.csv" file obtained from KEGG on April 17, 2018. These map KEGG IDs to KEGG pathway names.
